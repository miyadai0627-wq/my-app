# プロジェクトルール

## レスポンシブデザイン（必須）

**すべてのUI実装はレスポンシブデザインにすること。**

- モバイル（375px）・タブレット（768px）・デスクトップ（1280px）の3サイズで崩れないこと
- `@media (max-width: 600px)` を使ってスマホ向けのレイアウト調整を行う
- 横並びの要素はスマホでは縦積みに切り替える（`flex-direction: column`）
- 幅固定（`px`指定）ではなく `%` や `max-width` を基本とする
- 実装後は必ずモバイル・タブレット・デスクトップの3サイズでスクリーンショットを撮って確認する
